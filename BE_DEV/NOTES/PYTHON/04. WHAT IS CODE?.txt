WHAT IS CODE?Code is essentially nothing else than text in a file.How do you run code?You have some lines of code in a file. How do you run it?If you have a Python file (.py) or a JavaScript file (.js) for example, you can run them in your Terminal. Go in your Terminal to the directory of the file. Once you are in that directory, you have to use python after your prompt, plus the name and extension of the Python file you want to run. When you press Enter, the output of the lines of code will be printed in Terminal. To run the JavaScript file, you use node instead of python.How do you write code?You write code with a code editor. VSCode is the most commonly used editor program at the moment, 2022.Multiple filesNot all code has to be in one file. Multiple files can be connected to run a code together.
