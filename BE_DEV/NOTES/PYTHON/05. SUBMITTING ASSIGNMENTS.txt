(video)* Create things* Save things* Edit things* Save the thing againVersion Control assists in the last step: Save the thing again (and again, etc.). When, why and what you changed in the last edit. For a single user this is all pretty straight forward. Where Git shines is the collaborative history tracking. When multiple people work together on the same or various files and keep track of all modifications. Most importantly Git allows to Merge modifications done by various users.* Git is a fast and modern implementation of version control* Git provides a history of content changes* Git facilitates collaborative changes to files* Git is easy to use for any type of knowledge worker, not just for codersGit Basics: Get Going with Git (video)Download a Git installer.http://git-scm.com/downloads
